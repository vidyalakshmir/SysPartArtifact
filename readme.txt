This repository contains the implementation and evaluation of our work 'SysPart : Automated Temporal System Call Filtering for Binaries' (https://dl.acm.org/doi/10.1145/3576915.3623207) which has been presented at ACM CCS 2023. The paper is included in the directory paper/ within this repo.

This repository contains the artifact of SysPart that can be used to replicate the results in the paper. If you are looking for just the source code of different components of SysPart, please refer to this repository, which has detailed explanation of running the different components.
